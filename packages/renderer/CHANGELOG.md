# Changelog

## 0.1.0 (2026-04-14)

Initial release.

- Component registry with `registerComponent()`, `resolveComponent()`, `getRegisteredTypes()`
- 14 built-in components: Text, Markdown, Button, Image, TextInput, Icon, Divider, Container, CalendarModule, ChatModule, NotesModule, InputBar, Form, ScreenOptions
- `PageRenderer` for rendering full SDUI V2 pages
- Preset system via `registerPreset()` for swapping UI libraries
- React Native Paper preset (Button, Text, TextInput, Divider, Icon, Container)
